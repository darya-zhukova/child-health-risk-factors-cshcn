# child-health-risk-factors-cshcn
Data analysis project exploring child health risk factors and special health care needs using SQL, Python, weighted analysis, and logistic regression.

## Project Files

- `notebooks/child_health_risk_factors_analysis.ipynb` — final Python notebook with data cleaning, exploratory analysis, statistical testing, and logistic regression modeling.
- `sql/` — cleaned BigQuery SQL queries used for data preparation and analysis.
- `presentation/` — final capstone presentation.
- `visuals/` — selected charts and screenshots from the analysis.
